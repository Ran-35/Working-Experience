# 技术方案 及 设计文档 
### 层级编号  
<p style="text-indent: 2em;">
技术方案及设计文档，可以使用纯数字编号方式，也可以使用数字和字母组合的方式
</p>

```
1 系统架构（一级）
  1.1 硬件设计（二级）
    1.1.1 充电模块（三级）
      a) 功率单元（四级）
      b) 控制单元
  1.2 软件设计
```
```  
1 系统架构（一级标题）
1.1 硬件设计（二级标题）
1.1.1 充电模块（三级标题）
a) 功率单元（四级标题）
b) 控制单元（四级标题）
1.2 软件设计（二级标题）
```
### 字体规范  
```
1 [一级标题]  → 黑体/16pt/加粗
1.1 [二级标题] → 宋体/14pt/加粗
1.1.1 [三级标题] → 楷体/12pt
a) [四级标题] → 仿宋/11pt
```
### 间距设置  
```
一级标题前：1.5倍行距
一级标题后：1倍行距
---------------------
二级标题前：0.5倍行距
二级标题后：0.25倍行距
---------------------
三级标题与正文间：0行距
```
