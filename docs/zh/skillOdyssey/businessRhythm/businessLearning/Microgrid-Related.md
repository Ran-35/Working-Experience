# 微电网相关术语

## EMS系统相关概念
在微电网 (Microgrid) 中，EMS 通常指的是 能源管理系统 (Energy Management System)。它是微电网中的一个核心系统，负责监控、控制和优化微电网内的能源生产、存储、分配和消耗，以实现高效、可靠和可持续的能源管理。  
EMS 的主要功能包括：  
- 优化经济性和可持续性
- 实时监控与数据采集
- 故障检测与自愈功能
- 能源优化与调度
- 并网与离网控制
- 储能系统管理
- 负载管理
- 需求响应

### 并网与离网
1. **并网：微电网或分布式能源系统与主电网（公共电网）连接，并能够双向流动电力。**
- 电力双向流动
- 电网稳定性
- 经济性

2. **离网：微电网或分布式能源系统在不依赖外部电网的情况下独立运行。**
- 独立性
- 能源自给自足
- 储能依赖

3. **并离网切换：微电网能够根据电网状况、能源需求和外部条件的变化，在并网模式和离网模式之间根据监控、策略自动切换。**
- 平稳过渡
- 智能控制

### 微电网运行模式
*定义了微电网的运行状态或工作环境*
1. **孤岛模式**  
微电网完全独立于外部电网运行，所有的电力供给、分配和使用都由微电网内部的能源资源（如太阳能、风能、电池等）自行满足。
- 独立性：不依赖外部电网，所有的电力需求由本地的可再生能源、储能设备等自给自足。
- 能源储备：通常会有储能设备（如电池）以备不时之需。
- 自我调节：微电网内部需要精确的控制系统来管理能源生产、存储和消费，确保电力供应不间断。

2. **并网模式**  
微电网与公共电网连接，能够双向流动电力：在电力过剩时可以向电网输出多余电力，电力不足时则从电网获取补充电力。
- 双向电力流动：微电网与公共电网进行电力交换。
- 电网稳定性：可以利用公共电网的电力作为后备，增加系统的可靠性。
- 经济性：根据电网电价波动调节电力供给和需求，优化运营成本。

3. **混合模式**  
微电网能够在并网和孤岛模式之间切换，根据不同的电力需求和电网状况，选择最合适的工作模式。
- 切换能力：能够根据外部电网状况、负荷需求或可再生能源发电情况在并网和孤岛模式之间自动切换。
- 智能控制：需要复杂的控制系统和能源管理系统（EMS）来实时监控电网状态和微电网的负荷需求，做出最优决策。

4. **微网模式**  
微电网运行在不完全依赖外部电网的情况下，但不一定是孤立的。微网模式下，电网连接仍然存在，但微电网可以独立运行一段时间。
- 局部电力供给：适合大中型园区、企业等自给自足或部分依赖电网的用户。
- 稳定性：微电网模式下可以通过更精确的能源调度来减少对外部电网的依赖，提升供电稳定性。

5. **虚拟电厂模式**  
集中管理和调度多个分布式能源资源（如微电网、家庭能源系统等）来模拟一个大规模的电厂。通过集中的控制系统，多个小型微电网或分布式电源共同协调、共享电力资源。
- 分布式能源资源协调：多个分布式微电网或可再生能源设备通过虚拟电厂系统统一调度，以达到类似于传统电厂的规模效益。
- 负荷平衡：在电网负荷过高时，虚拟电厂可以调节电力输出，提供电力支援。
- 市场参与：虚拟电厂可以参与电力市场，通过提供备用电源、调节负荷等方式获得经济效益。

6. **分布式能源系统模式**  
微电网中的发电设备（如太阳能、风能、燃气轮机等）分散部署在电网的各个节点，而非集中在某一地点。通过这种方式，微电网能在分散的地点生成电力并供给相应区域。
- 分布式资源：通过多个小型发电设备生成电力，减少了电力传输损耗。
- 灵活性和冗余性：多个发电源可以相互补充，增强系统的可靠性。
- 可再生能源集成：通常使用太阳能、风能等可再生资源来进行分布式发电，增加系统的绿色能源比例。

7. **微电网集群模式**  
多个微电网系统在某一地区进行联合运作，通过一个集中管理的系统来协调各个微电网之间的电力调度和负荷管理。
- 集群协同：多个微电网之间相互协作，可以分享能源资源，提高运行效率。
- 负荷分担：通过集群管理，可以合理分配电力负荷，减少单个微电网的压力。

### 微电网的稳定性
电网的稳定性与电压和频率密切相关，它们反映了电网运行的健康状况，且互相影响。
1. **频率**  
频率反映了电网中发电和负载之间的平衡。电网频率的波动通常反映了系统内有功功率（P）的平衡状态。
- 负载增加（例如，更多的设备接入电网），而发电功率保持不变时，频率会下降。
- 发电增加（例如，发电机输出增加），而负载保持不变时，频率会上升。
- 频率过低：表示系统的有功功率不足，可能会导致设备无法正常运行，甚至发电机脱网或停运。
- 频率过高：表示有功功率过剩，可能会导致发电机过载或电力设备损坏，严重时可能导致系统崩溃。

2. **电压**  
- 无功功率不足：当电网的无功功率不足时，电压会下降。例如，大型感性负载增加时（如大型电动机启动），可能会导致电压瞬时下降。
- 无功功率过剩：当电网中的无功功率过多时，电压会上升。这通常发生在静态电容器过多或电网过于“松弛”时。
- 电压过低：电压下降会导致电气设备无法正常工作，电动机可能失速，变压器可能过热，甚至可能导致设备停运或损坏。长期电压不足会导致用电设备不可靠运行，影响工业生产和生活质量。
- 电压过高：电压过高可能会损坏设备，尤其是对于高压设备（如变压器、电力电子设备），它们对电压的容忍度有限。
### 微电网控制策略
*动态调节微电网系统行为的方式。定义了在不同工作模式下，如何根据不同的控制目标（如功率、频率、电压等）来调节和管理微电网中的发电单元、储能设备和负荷。*

1. **PQ模式 (有功功率和无功功率控制模式 Power-Quality Control Mode)**  
PQ模式主要用于微电网中 无功功率（Q） 和 有功功率（P） 的控制，通常是在并网状态下使用。 
- 特点：在PQ模式下，微电网控制器设定系统需要的有功功率（P）和无功功率（Q），并确保发电设备的输出能达到这个设定值。
- 特点：该模式适用于电网中能量流动相对稳定的场合，控制的重点是 功率平衡，尤其是无功功率的调节。
- 应用：微电网并网运行时，控制发电机组（如光伏、风机、储能系统等）输出与电网相匹配的有功和无功功率。
- 作用：匹配电网的负载需求，确保频率（有功功率控制）和电压稳定（无功功率控制）。

2. **VF模式 (电压频率控制 Voltage-Frequency Control Mode)**  
VF模式是通过控制 电压（V） 和 频率（f） 来稳定微电网的运行。
- 特点：在VF模式下，控制器根据电压和频率的变化调节发电机组输出，使得微电网的电压和频率保持在稳定的范围内。
- 特点：它是一种基于系统的整体运行来调节负载和发电的控制策略。
- 应用：通常用于微电网离网模式，尤其是当电网频率和电压需要严格保持时。

3. **VSG模式 (虚拟同步发电机模式 Virtual Synchronous Generator Mode)**    
VSG模式模仿同步发电机的特性，通过模拟 同步发电机的惯性 和 阻尼特性 来提高微电网的稳定性。
- 特点：通过控制器模拟同步发电机的物理惯性、阻尼特性和电气特性，改善系统在负荷波动、风速变化等情况下的稳定性。
- 特点：VSG可以通过模拟同步发电机的转动惯性来提供 短期频率调节，缓解频率的波动。
- 特点：使得微电网中的 非同步发电源（如光伏、风机）在并网时能够像同步发电机一样稳定。
- 应用：VSG模式通常用于包含大量可再生能源的微电网，尤其是在离网运行时，它可以模拟传统发电机的惯性特性，增强系统的抗扰动能力。  

**问**：为什么同步发电机可以可以维持频率？  
**答**：发电机和电网是同步运行的，通常是通过调节电网的频率来平衡发电机的输出功率与系统负荷。系统中所有发电机的转子在频率稳定时会以相同的速率旋转。当电网的频率发生波动时，系统中的发电机转速会随之发生变化。  

4. **Droop控制模式 (Droop Control Mode)**    
Droop控制是一种频率-功率控制技术，它通过控制 频率 或 电压 来实现发电机输出功率的调节，广泛应用于并网模式和孤岛模式中。
- 特点：频率-功率 droop：根据发电机输出的频率变化来调节有功功率。频率下降时，发电机增加功率输出，频率上升时则减少功率输出。该控制方式能自动适应负荷的变化，避免频率过低或过高。
- 特点：电压-功率 droop：通过电压变化来控制无功功率输出。电压下降时，发电机增加无功功率输出，电压上升时则减少无功功率输出。
- 应用：在 并网控制 和 孤岛运行模式 中都能使用Droop控制。它不需要精确的通信或同步信号，而是依赖系统中每个发电单元的“droop”特性，适合分布式发电和储能系统。

5. **P-V控制模式 (Power-Voltage Control Mode)**    
P-V控制模式主要通过控制**有功功率（P）和电压（V）**的关系，保持微电网的稳定性。
- 特点：在P-V控制模式中，控制器通过调整发电机的输出功率（P）来维持系统的电压（V）稳定。
- 特点：该模式通常适用于微电网并网操作，尤其是光伏和风电发电系统中，能够调节输出功率以适应电压变化。
- 应用：该控制模式广泛用于微电网并网时调节发电机组的功率输出，以保持电网电压稳定。

## 设备工作模式
#### **光伏（Photovoltaic, PV）**
1. 最大功率点追踪（MPPT）模式  
光伏系统中的逆变器通过MPPT算法实时调整工作电压和电流，确保光伏面板始终在最佳功率点运行，最大化发电效率。

2. 并网模式  
光伏系统与电网连接，实时向电网输出电力。光伏发电超过负荷需求时，剩余电力可并入电网。

3. 离网模式  
光伏系统独立于电网运行，常与储能系统搭配使用。适用于无电网的区域或特定情况下（如紧急供电）。

4. 并离网自动切换模式  
系统能够根据电网的状况切换至并网或离网模式，例如电网断电时，系统自动转为离网模式供电  

#### **储能（Energy Storage, ES）**
1. 充电模式  
储能设备（如电池、超级电容器）在电力需求低或有富余电力时充电，存储多余的电能。

2. 放电模式  
储能设备在电力需求高或电网供电不足时，向电网或负荷提供电力。

3. 峰谷调节模式  
储能系统利用电价差异，在电网电价较低时充电，电价较高时放电，从而降低运营成本。

4. 备用电源模式  
在电网故障或停电情况下，储能系统作为备用电源提供持续电力。

#### **充电桩（EV Charger）**  
1. 普通充电模式  
电动汽车直接通过充电桩充电，电流和电压按照预设标准稳定提供。

2. 快充模式  
充电桩提供较大的充电功率，使电动汽车在较短的时间内完成充电。

3. 智能充电模式  
结合用户需求、环境条件、电网负荷等因素动态调节充电速率。例如，通过负荷平衡或调度系统来避免电网压力过大。

4. V2G（Vehicle-to-Grid）模式  
充电桩与电动汽车的电池双向互动，电动汽车的电池可以反向向电网提供电力，参与电网调节或平衡负荷。

#### **风机（Wind Turbine）** 
1. 最大功率点追踪（MPPT）模式  
风机通过控制叶片角度，实时调整风机的转速，使其始终工作在最大功率输出点。

2. 并网模式  
风力发电机将发电电力直接并入电网，补充电网负荷。

3. 孤岛模式  
在电网断电时，风力发电机可以独立供电，尤其与储能设备搭配，提供局部电力。

4. 风速控制模式  
风机根据风速变化自动调节其转速和功率输出，以确保稳定发电并防止过载。

5. 低风速模式  
风速较低时，风机可能进入低速运行或待机状态，等待风速上升。

#### **柴油发电机（Diesel Generator, DG）** 
1. 常规发电模式  
柴油发电机根据负荷需求提供持续电力，通常用于远程或电网不稳定的地区。

2. 备用模式  
在电网停电或负荷过高时，柴油发电机作为备用电源启动，提供电力。

3. 并网模式  
当电网电力需求超出供应时，柴油发电机与电网并网提供电力，辅助电网运行。

4. 负荷跟踪模式  
根据负荷的变化自动调整发电功率，以确保供电平稳和效率最优。

#### **负荷（Load）** 
1. 定常负荷模式  
负荷保持稳定的电力需求，通常用于家庭、办公室等日常用电设备。

2. 可调负荷模式  
根据电网需求，负荷设备（如空调、加热器等）可以在不影响用户使用的情况下调节用电量，例如在电力紧张时适当降低功率。

3. 峰谷负荷模式  
负荷会根据电价、时间等因素变化。例如，电力需求较低时，某些设备可以在低电价期间运行，减少电费支出。



## 常见问题
**问**：微电网中，储能系统为什么不能充满？要保持一定余量？  
**答**：(1)保持一定的余量，避免电池长期充满或者完全放电，能显著延长电池的使用寿命。(2)储能系统应能够适时地释放或储存电能，以维持电网的稳定性
