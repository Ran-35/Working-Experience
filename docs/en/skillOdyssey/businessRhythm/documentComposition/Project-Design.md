# 项目书 及 设计方案  
### 层级编号  
<p style="text-indent: 2em;">
项目书及设计方案中，分层级序号应遵循国家规范《GB/T 15835-2011 出版物上数字用法》和公文写作标准  
</p>

```md
一、第一级标题（章）
    （一）第二级标题（节）
        1. 第三级标题（条）
            (1) 第四级标题（款）
                ① 第五级标题（项）
                    a. 第六级标题（目）
```
<br>

| 层级   | 序号样式           | 使用场景                         | 排版要求                 |
| ------ | ------------------ | -------------------------------- | ------------------------ |
| 第一级 | 一、二、三...      | 核心章节（如项目概述、技术方案） | 黑体/小二号，段前段后1行 |
| 第二级 | （一）（二）（三） | 子模块划分                       | 楷体/三号，段前段后0.5行 |
| 第三级 | 1、2、3....        | 具体技术点                       | 仿宋/小三，首行缩进2字符 |
| 第四级 | (1) (2) (3)...     | 技术参数/功能点                  | 仿宋/四号，首行缩进4字符 |
| 第五级 | ① ② ③...           | 补充说明项                       | 仿宋/小四，首行缩进6字符 |
| 第六级 | a. b. c....        | 列表项/备注项                    | 仿宋/小四，首行缩进8字符 |

### 图表/公式编号  
```
图1-1（第一章第一个图）
表2-3（第二章第三个表）
公式(3-4)（第三章第四个公式）
```

### 附录编号  

```
附录A ×××技术参数 
附录B ×××认证证书
```

### 法律条款引用

```
根据《XXX法》第5条第2款规定...
```

### 禁止使用的序号形式

```
❌ 1)、2)、3)（带右括号的阿拉伯数字）
❌ 1.1、1.1.1（多级点号组合）
❌ ◆ ■ ●（项目符号代替层级）
```

### 标题格式

```
前三级必须带标题文字（禁止单独使用序号）
第四级后可省略文字（如直接列技术参数）
```

### 跨页处理

```
禁止二级标题单独出现在页末（至少带1行正文）
超过5行的大段内容需分页时添加（续）
```


### 具体示例
```
一、项目概述
    （一）建设背景
        1. 政策依据
            (1) 《十四五规划纲要》
                ① 新能源发展目标
                    a. 充电桩覆盖率≥80%
            (2) 地方补贴政策
        2. 市场需求
    
二、技术方案
    （一）系统架构
        1. 硬件组成
            (1) 充电模块
                ① 120kW直流桩
                    a. 双枪动态分配
            (2) 监控系统
        2. 软件设计
```
